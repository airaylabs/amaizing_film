# raymAIzing film - Google Opal Flows

Folder ini berisi **32 Opal flow definitions** untuk setiap tool di raymAIzing film.

## Cara Menggunakan

1. Buka [Google Opal](https://opal.google)
2. Klik "Import Flow" atau "Create New Flow"
3. Upload file JSON dari folder ini
4. Flow akan tersedia di Opal Anda
5. Copy link flow dan paste ke Admin Global Links di raymAIzing film

## AI Models Used

- **Gemini 2.0 Flash** - Text generation, analysis, planning
- **Imagen 3** - Image generation
- **VEO 3** - Video generation
- **MusicLM/AudioGen** - Audio generation
- **Google TTS Wavenet** - Voice synthesis

## Daftar Flows (32 Total)

### Pre-Production (01-04)
- `01-script-to-treatment.json` - Convert script to visual treatment
- `02-storyboard-creator.json` - Generate storyboard frames
- `03-character-designer.json` - Design characters
- `04-world-builder.json` - Design locations/environments

### Production - Image (05-07)
- `05-text-to-image-pro.json` - Generate images from text
- `06-character-transform.json` - Transform character photos
- `07-scene-generator.json` - Generate scene backgrounds

### Production - Video (08-11)
- `08-text-to-video-pro.json` - Generate video from text (VEO 3)
- `09-image-to-video-pro.json` - Animate images (VEO 3)
- `10-dialogue-animator.json` - Animate talking characters
- `11-action-sequence.json` - Generate action scenes

### Post-Production (12-14)
- `12-video-extender.json` - Extend video clips
- `13-transition-maker.json` - Create scene transitions
- `14-color-mood-matcher.json` - Match colors across scenes

### Distribution (15-16)
- `15-thumbnail-generator.json` - Create thumbnails
- `16-poster-promo.json` - Create posters & promo materials

### Ideation (idea-01 to idea-03)
- `idea-01-trend-explorer.json` - Discover viral trends
- `idea-02-idea-generator.json` - Generate story ideas
- `idea-03-genre-mixer.json` - Mix genres for unique concepts

### Story Development (story-01 to story-04)
- `story-01-synopsis-writer.json` - Create master synopsis
- `story-02-episode-breakdown.json` - Break into episodes
- `story-03-scene-planner.json` - Plan scenes
- `story-04-character-arc.json` - Define character journeys

### Production - Audio (audio-01 to audio-04)
- `audio-01-dialogue-generator.json` - Generate voice lines
- `audio-02-background-music.json` - Create scene music
- `audio-03-sound-effects.json` - Generate SFX
- `audio-04-audio-mixer.json` - Mix audio layers

### Post-Production Extras (post-01 to post-03)
- `post-01-scene-assembler.json` - Combine scenes
- `post-02-viral-picker.json` - Pick viral moments
- `post-03-export-manager.json` - Export settings

### Distribution Extras (dist-01 to dist-02)
